<h1 align="center">Img2Ascii</h1>

## Requirements

```
sudo apt-get install libsdl2-dev &&
sudo apt-get install libsdl2-image-dev -y
```

## How does this work?

Compile with:

```
chmod +x compile.sh
./compile.sh
```

And run:

```
./exec [PATH/2/IMG]
```

You can save it to a file like this:

```
./exec [PATH/2/IMG] > art.txt
```
